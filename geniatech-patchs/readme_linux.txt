1.get the source supplied by Amlogic:    http://openlinux.amlogic.com/download/linux/ARM/kernel/2.change the mkimage:a. backup arch/arm/boot/mkimageb. replace it with new mkimage.3.build the uImage:copy the following to the path arch/arm/mach-meson with replace option. Kconfig Makefile stv-mbx-m3.c stv-mbx-m3.hcopy config.config as .config to the kernel root directory and "make uImage"4.burn uImage into board:copy uImage file from arch/arm/boot/uImage to SDCARD.enter uboot and input cmd:mmcinfo;fatload mmc 0 82000000 uImage;nand erase 800000 400000;nand write 82000000 800000 400000;5.burn your rootfs (must be gaffs format):mmcinfo ; fatload mmc 0 0x82000000 rootfs; nand erase 0x000000c00000; nand write 0x82000000 0x000000c00000 0x000010c00000
